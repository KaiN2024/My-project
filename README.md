# Реализованные механики

На данный момент присутствует простое меню с объяснением управления.
Все действия персонажа и противников с анимированы.
Реализована механика движения по поверхностям уровня.
Сделан простой ИИ для противников, перемещение в определенной области и нанесение урона персонажу игрока если тот подойдет слишком близко.
Присутствуют механики двойного прыжка и сприта.
Персонаж может атаковать противников и получать от них урон, уровень здоровья персонажа отображается в соответствующих иконках, реализована механика восполнения здоровья посредством размещенных на уровне предметов.
Есть возможность ставить игру на паузу.

# Уровни

На данный момент построен один уровень с двумя этажами, присутствует механика возврата персонажа до контрольной точки при падении в пропасть и в начало уровня при смерти.
Также на этажах расположены статичные и динамические платформы для преодоления их игроком.
Игрок способен перемещаться между этажами посредством телепортов, расположенных на уровне.
